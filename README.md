# Vite + Tailwind CSS starter + ZIPE

Note if you have access to [Tailwind UI](https://tailwindui.com), you can use [this version instead](https://github.com/posva/vite-tailwind-starter/tree/tailwind-ui).

## Installation

```sh
yarn
```

## Development

```sh
node ./dev.js
```

## Build

```sh
yarn build
```
